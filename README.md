## Moving to Codeberg.org !
*MechatronicBeing* is migrating to [Codeberg.org](https://codeberg.org/cyberus) !  
Codeberg.org is a democratic community-driven, non-profit, privacy-friendly, software development platform.   

## The "Cyber-Us" project, MechatronicBeing's evolution
A new organisation, [**cyberus**](https://codeberg.org/cyberus), was created on codeberg.org, and replace *MechatronicBeing*.  
The **Free "public domain" resources, for anyone and any purpose** will be **added gradually** in [cyberus](https://codeberg.org/cyberus).  
A new system will be created, to ease the process with the public domain works.

## The Cerberus's "Cyber-Us" guide : General intelligences in machines
[**CERBERUS Ace**](https://codeberg.org/cerberus) will continue the main MechatronicBeing objective : publishing a **free public guide for general intelligences in machines**.
With the importance and the growing need of human-like behaviours in machines, the writing of this guide is a **top priority**, with regular updates.  
This guide is written [here](https://codeberg.org/cerberus/cyberus-guide), in French at the moment.
