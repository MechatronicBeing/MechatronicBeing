## Moving to Codeberg.org !
*MechatronicBeing* is migrating to [Codeberg.org](https://codeberg.org/cyber-us) !  
Codeberg.org is a democratic community-driven, non-profit, privacy-friendly software development platform.   
A new organisation, [Cyber-Us](https://codeberg.org/cyber-us), was created.  

## the "Cyber-Us" project, by Ace
The member **Ace** will continue the main MechatronicBeing project : a complete guide for bots and robots with a human-like artificial intelligence.  
With the importance and the growing need of human-like programs and machines, the writing of this guide is a priority, with regular updates.  
The new project is hosted [here](https://codeberg.org/cerbere/cyber-us), written in French, at the moment, by [Ace & Rex 'CERBÈRE'](https://codeberg.org/cerbere).  

## The old MechatronicBeing resources
The *"Free 'public domain' resources, usable by Humans and Machines"* will come back, step by step, on [Codeberg](https://codeberg.org/cyber-us).  
A new system will be created, to ease the process (found, download and manage the public-domain works, with links to theirs authors).
